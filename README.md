# TG_BOT_template

創建TG機器人，並生成TOKEN 
放置在 同資料夾下 創建 token.txt 放入裡面
即可做使用

# 創建機器人
:::info
在 Telegram 搜尋 `@BotFather` 並開始對話。
    
發送 `/newbot` 指令。
    
依照提示輸入：
    
1. Bot 名稱（可見名稱）
        
1. Bot 使用者名稱（必須以 `bot` 結尾，如 `MyTestBot`）
        
完成後會收到一組 **Token**，這是你的 Bot API 金鑰。
:::

# 如需開啟 內連模式
:::success
打開 Telegram 找到 @BotFather
輸入指令 
>/setinline

選擇你的機器人（Bot）
根據指示，BotFather 會要求你輸入在用戶輸入 @bot 時的提示內容。
:::
